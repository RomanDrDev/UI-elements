Модальные окна (CSS, псевдокласс :target || JS)

Модальные окна, их еще называют всплывающие окна или попапы от английского (pop-up) - блоки с версткой находятся перед закрывающимся тегом body. Используем якорные ссылки и анимацию.

Два варианта работы - открытие с помощью CSS псевдокласса :target и, более мощный, с использованием JavaScript.

Источник: Модальные окна(попапы) на HTML CSS JS

Простые модальные окна

Преимущества

  1. Простая реализация
  2. Удобно для показа информационных окон
  3. Связываем кнопки с окнами через data-атрибут

Недостатки

  1. Закрытие только по клику на оверлей
  2. Не подходит для попапов внутри попапов
Источник: Попапы на HTML CSS JS

Попап открыт с помощью CSS псевдокласса :target

Преимущества

  1. Простая реализация, только HTML и CSS
  2. Удобно для показа информационных окон

Недостатки

  1. При закрытии окна происходит скачок к началу страницы (решение: сделать объект, н-р, шапка с меню, либо добавить невидимый объект, фиксированным, и ссылку указать на них)
  2. Нельзя закрыть при клике на оверлей (решение: создать дополнительную ссылку (popup__area), родителем которой будет попап)
  3. В адресной строке сохраняется хеш
  4. Нельзя зафиксировать контент под областью попапа и оверлея
  5. Нельзя скрыть боковой скрол (при переполнении содержимого) и убрать скачок в сторону

Попап открыт с помощью JS

Преимущества

  1. Допускается наличие вложенных попапов
  2. Фиксация/блокировка контента под областью попапа и оверлея
  3. Скрываем боковой скрол, предотвращаем сдвиг в сторону
  4. Широкий функционал по анимации, закрытие по любому элементу с классом close-popup, оверлею либо клавишей Esc

Недостатки

  1. Требуется код JS
